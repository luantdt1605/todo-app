## Installation

1. Clone the project:

```bash
git clone https://github.com/luantdt1605/ai-assistant.git
cd todo-app
```

2. Install dependencies:

```bash
npm install
```

3. Run the application in development mode:

```bash
npm start
```

The application will run at [http://localhost:3000](http://localhost:3000)
